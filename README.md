# yt-summarizer-bot
# 🤖 AI YouTube Summarizer Bot

![Bot Status](https://img.shields.io/badge/Status-Live-brightgreen)
![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Gemini AI](https://img.shields.io/badge/AI-Google%20Gemini-orange)

Ek advance Telegram bot jo YouTube videos ko analyze karke unka **Hinglish Summary** nikaal deta hai. Ye bot Render ke free tier par optimized chalne ke liye design kiya gaya hai.

---

## ✨ Features

- 🚀 **Fast Summarization:** Kuch hi seconds mein video ka nichod.
- 🗣️ **Hinglish Support:** Summary aisi language mein jo sabko samajh aaye.
- 📱 **Interactive UI:** Buttons aur progress alerts ke saath professional experience.
- ⚡ **Lightweight:** Low RAM usage (Render Free Tier compatible).
- 🛠️ **Status Commands:** Bot ki health check karne ke liye `/status` aur `/help`.

---

## 🛠️ Tech Stack

- **Language:** Python 🐍
- **Framework:** `python-telegram-bot`
- **AI Model:** Google Gemini Pro 🤖
- **API:** YouTube Transcript API 📺
- **Hosting:** Render (Web Service) ☁️

---

## 🚀 Setup & Installation

1. **Clone the Repo:**
   ```bash
   git clone [https://github.com/ayuuu1233/yt-summarizer-bot.git](https://github.com/ayuuu1233/yt-summarizer-bot.git)
2. Install Dependencies:
   pip install -r requirements.txt

3. Set Environment Variables:
​TELEGRAM_TOKEN: BotFather se mila hua token.
​GEMINI_API_KEY: Google AI Studio se mili hui key.

4. Run the Bot:
   python main.py

📂 Project Structure
​main.py: Bot ka main logic aur commands.
​keep_alive.py: Flask server jo bot ko hosting par active rakhta hai.
​requirements.txt: Saari zaroori libraries ki list.
