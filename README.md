## ⚙️ Setup & Deploy

### 1️⃣ Clone the Repo
```bash
git clone https://github.com/yourusername/party-riot-bot.git
cd party-riot-bot
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Environment Variables Set Karo
`.env` file banao ya Render pe directly add karo:

```env
TELEGRAM_TOKEN=your_bot_token_here
GEMINI_API_KEY=your_gemini_api_key_here
OWNER_ID=your_telegram_user_id
```

### 4️⃣ Run Karo Locally
```bash
python party_bot.py
```

---

## 🚀 Deploy on Render

```
1. GitHub pe repo push karo
2. render.com pe "New Web Service" banao
3. Repo connect karo
4. Environment variables add karo
5. Build Command  → pip install -r requirements.txt
6. Start Command  → python party_bot.py
7. Deploy! 🎉
```
